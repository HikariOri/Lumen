# 四、Input System（核心系统）

这是整个架构的“翻译器”。

---

## ✔ 1. SDL → Raw Input

```cpp
void processSDL(SDL_Event& e, RawInputState& input) {
    if (e.type == SDL_EVENT_KEY_DOWN)
        input.keys[e.key.keysym.scancode] = true;

    if (e.type == SDL_EVENT_KEY_UP)
        input.keys[e.key.keysym.scancode] = false;
}
```

---

## ✔ 2. Raw → Action Mapping

```cpp
class InputSystem {
public:
    std::unordered_map<SDL_Scancode, Action> map;

    ActionState actionState[10];

    void update(const RawInputState& input) {
        for (auto& [key, action] : map) {

            bool isDown = input.keys[key];

            auto& state = actionState[(int)action];

            state.pressed  = isDown && !state.held;
            state.released = !isDown && state.held;
            state.held     = isDown;
        }
    }
};
```

# 七、完整 Frame 顺序（必须这样）

```cpp
void frame() {

    // 1. SDL poll
    while (SDL_PollEvent(&e)) {

        ImGui_ImplSDL3_ProcessEvent(&e);

        if (!ImGui::GetIO().WantCaptureKeyboard)
            processSDL(e, inputState);
    }

    // 2. update input mapping
    inputSystem.update(inputState);

    // 3. emit semantic events
    emitEvents(inputSystem, dispatcher);

    // 4. process event queue
    dispatcher.update();

    // 5. gameplay systems react
}
```
