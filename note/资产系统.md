# 🔥 1. AssetID（核心）

---

## ❗ 不要用 string 做 key

👉 用：

```cpp
using AssetID = uint64_t;
```

---

## 生成方式：

```cpp
AssetID HashPath(std::string_view path);
```

---

👉 原因：

* 更快
* 可序列化
* 类似 Unity GUID

---

---

# 🔥 2. AssetHandle（非常关键）

---

```cpp
template<typename T>
struct AssetHandle
{
    AssetID id;
};
```

---

👉 ECS / Renderer 只拿 handle：

```cpp
AssetHandle<Mesh>
AssetHandle<Texture>
```

---

---

# 🔥 3. AssetManager（核心系统）

---

```cpp
class AssetManager
{
public:
    template<typename T>
    AssetHandle<T> Load(std::string_view path);

    template<typename T>
    T* Get(AssetHandle<T> handle);

private:
    std::unordered_map<AssetID, std::shared_ptr<void>> assets;
};
```

---

---

# 🔥 4. Loader 分离（非常关键）

---

```cpp
class IAssetLoader
{
public:
    virtual void* Load(std::string_view path) = 0;
};
```

---

## 实现：

```cpp
class GltfLoader : public IAssetLoader {};
class TextureLoader : public IAssetLoader {};
```
